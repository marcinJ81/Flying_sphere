### Flying sphere
How throw sphere.
#### Instruction:
- 1 (button) Main Camera
- 2 (button) Second Camera
- Up arrow - increases velocity
- Down arrow - decreases velocity
- Left arrow - decreases angle
- Right arrow - increases angle
